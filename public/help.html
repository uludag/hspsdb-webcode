Indexing your BLAST results

HSPs-db-indexer project can help you with initializing an Elasticsearch index and indexing your BLAST results.

Installing HSPs-db web interface

Source code of HSPs-db web interface is maintained by the HSPs-db-webcode project that could be used to build web interfaces for indexed BLAST results.
