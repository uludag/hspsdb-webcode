HSPs-db: Database for sequence similarity search results

HSPs-db project aims to develop data mining interfaces and indexing tools for NCBI-BLAST search results. The web site you are reading this note is linked to an Elasticsearch database that contains sample BLAST results.

Help page includes information for indexing BLAST results and installing this web interface for indexed BLAST results.
